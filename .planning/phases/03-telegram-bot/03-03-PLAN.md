|----------|-------------|
| Telegram network -> bot handlers | External user updates/callbacks cross into backend handlers |
| Bot handler -> graph runtime | Handler-provided state crosses into policy engine |
| Bot callback -> feedback DB | User input can trigger persistence path |

</threat_model>
