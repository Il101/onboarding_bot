<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Uploaded PDF -> Docling parser | Untrusted file potentially containing malicious content |
| Uploaded .ogg -> Groq Whisper API | Audio data sent to external API (may contain PII in voice form) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-01-08 | Tampering | PDF parser | mitigate | Validate file extension and content type before passing to Docling. Limit file size to 100MB per config |
| T-01-09 | Information Disclosure | Groq Whisper API | accept | Voice recordings may contain PII in audio form. Groq processes audio externally. Mitigation: voice data is transcribed BEFORE PII anonymization, but the raw audio is not stored. Accept risk: Groq is a transient API, data not persisted there. D-07 ensures text PII is anonymized after transcription |
| T-01-10 | Denial of Service | Groq API calls | mitigate | Handle rate limits and timeouts with retry logic (max_retries=3). Large files chunked to avoid timeouts |
</threat_model>
