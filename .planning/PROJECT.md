# V-Brain: AI Knowledge Extractor & Mentor

## What This Is

Система извлечения бизнес-знаний из рабочих чатов (Telegram) и документов (PDF), которая превращает разрозненную информацию в структурированную базу знаний (Wiki) и AI-ментора для новых сотрудников. Новые сотрудники задают вопросы через Telegram-бота и получают точные ответы на основе внутренних инструкций компании.

## Core Value

Новый сотрудник может получить ответ на любой рабочий вопрос за секунды, используя знания извлечённые из реальных коммуникаций и документов компании — без участия владельца или наставника.

- Мульти-тенант архитектура — разворачивается для одного бизнеса (но закладываем под масштабирование)
- OAuth и сложная авторизация — простой логин для MVP

## Context

- В малом и среднем бизнесе знания живут в головах владельцев и в рабочих чатах
- При увольнении ключевых сотрудников или росте команды знания теряются
- Владелец тратит время на однотипные вопросы новых сотрудников
- Есть готовый черновик спецификации (Spec_V-Brain_AI_Automation.pdf) от заказчика
- Язык системы — русский (данные и ответы)
- Данные должны оставаться в пределах периметра компании

## Constraints

- **Безопасность**: PII-анонимизация обязательна — ФИО, телефоны, email заменяются на токены перед обработкой
- **LLM**: Не привязан к конкретной модели — можно использовать API (OpenAI, Anthropic) или локальный деплой (Gemma, Qwen, Llama)
- **Источники MVP**: Только Telegram логи (JSON) и PDF — видео парсинг отложен
- **Периметр**: Данные не покидают периметр компании при локальном деплое LLM
- **Язык**: Все данные, извлечённые знания и ответы на русском
