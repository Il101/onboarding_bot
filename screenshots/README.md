## How to Add Screenshots

1. Take a screenshot of the relevant UI
2. Save as PNG/JPG in this directory
3. Update README.md with the screenshot path

Example:
```markdown
![Admin Panel](screenshots/admin-panel.png)
```

## Naming Convention

- `admin-panel-*.png` — Admin UI screenshots
- `bot-chat-*.png` — Telegram bot screenshots
- `architecture-*.png` — Architecture diagrams
